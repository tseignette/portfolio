Confidentialité

De manière générale, ce site ne stocke aucune donnée dans le navigateur (cookies, stockage local).

Hébergement

Le site est hébergé à l'aide de Github Pages. Il se peut que ce service collecte des données :

Please note that GitHub may collect User Personal Information from visitors to your GitHub Pages website, including logs of visitor IP addresses, to comply with legal obligations, and to maintain the security and integrity of the Website and the Service.

- GitHub Privacy Statement

Statistiques

Le site utilise Insights afin de collecter des statistiques relatives à l'utilisateur (langue, taille d'écran, navigateur, etc.) et à ses actions (pages vues, durée de la visite, clic sur certains éléments, etc.).
Insights est un service de statistiques axé sur la confidentialité. Il ne stocke aucune donnée dans le navigateur et respecte le paramètre Do not track du navigateur. Voir les données collectées
